# 🧩 Algorithm Problem Solving Archive

BaekjoonHub를 통해  
온라인 저지 플랫폼의 알고리즘 문제 풀이 코드를  
자동으로 업로드·관리하는 레포지토리입니다.

---

## 📁 Repository Structure
- 알고리즘 문제 플랫폼별로 폴더를 분리해 관리합니다.
  - `Baekjoon/`
  - `Programmers/`
- 플랫폼은 추후 추가될 수 있습니다.
- 각 문제의 풀이 코드는 해당 플랫폼 폴더 내부에 정리되어 있습니다.
- 하나의 문제에 대해  
  여러 접근 방식이나 개선된 풀이가 있을 경우  
  **복수의 풀이 코드가 저장될 수 있습니다.**

---

## 🗂 Unsolved Policy
- 해결하지 못한 문제는 `미해결` 폴더에 업로드됩니다.
- 단, **당일 내에 해결한 문제는 미해결 폴더에 남기지 않습니다.**
- `미해결` 폴더에는  
  해당 날짜 기준으로 끝까지 해결하지 못한 문제만 유지됩니다.
- 이후 문제를 다시 풀어 **성공(AC)** 하면  
  해당 미해결 코드는 삭제되고,  
  최종 풀이 코드만 플랫폼 폴더에 남습니다.

---

## 📝 Wrong Answer Notes
- 미해결 및 재도전 과정에서 발생한  
  사고 흐름과 실패 원인은 블로그에 정리합니다.
- 코드는 저장하지 않고,  
  **문제 해결 과정 중심으로 기록합니다.**
